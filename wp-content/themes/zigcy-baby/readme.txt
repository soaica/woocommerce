== Description ==
Zigcy Baby is a free eCommerce/WooCommerce child theme of Zigcy Lite which lets you create an online store in no time. The theme is perfect for selling baby products online or for creating any kind of kids related eCommerce sites. It is the most functional WordPress web shop theme integrated with powerful WooCommerce features. The theme has clean and sleek design with easy customization options.
